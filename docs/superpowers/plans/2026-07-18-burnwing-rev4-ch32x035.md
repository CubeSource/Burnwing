# Burnwing Rev 4 — CH32X035 PD MCU Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the HUSB238 PD sink IC with a CH32X035F8U6 MCU that negotiates USB-PD in firmware and adds current telemetry, USB CDC control, and an I2C slave interface, per `docs/superpowers/specs/2026-07-18-ch32x035-pd-mcu-design.md`.

**Architecture:** Two independent workstreams. **Hardware** (Tasks 1–9): KiCad schematic + layout changes in this repo. **Firmware** (Tasks 10–16): a PlatformIO project in `firmware/`, developed and tested against the WeAct CH32X035 Core Board (identical chip) before rev-4 boards exist, then bench-validated on real hardware. The MCU only ever *adds* drive to the burn channels (diode-OR); all existing hardware burn paths are untouched.

**Tech Stack:** KiCad 10 + `kicad-cli`, WCH CH32X035 EVT SDK (SPL peripheral library), PlatformIO community `ch32v` platform (fallback: MounRiver Studio), WCH-LinkE probe.

## Global Constraints

- Branch: `burnwing-eng-v4`. Commit after every task. No Claude attribution on commits.
- Never commit `.kicad_prl` or `.lck` files.
- Voltages: 5 / 9 / 12 / 15 V (indices 0–3). Blank/dead MCU must still yield a working 5V board.
- MCU burn-drive pins must be electrically incapable of firing a channel during power-up/reset/flash (Hi-Z at reset + existing pull-downs + diode-OR).
- I2C slave 7-bit address 0x42; burn over I2C requires arm (0xA5) → fire inside a 500 ms window.
- SWD boot window: pins PC18/PC19 stay SWD for 2000 ms after reset before remapping to I2C.
- Pin map (from spec — all firmware and schematic work must match):
  CC1/CC2=PC14/PC15 · USB D−/D+=PC16/PC17 · SWDIO~SDA=PC18, SWCLK~SCL=PC19 (`GPIO_PartialRemap3_I2C1`) · burn drive=PB3(ch1)/PB11(ch2) · IS sense=PA0(ch1)/PA1(ch2) · command taps=PA2(ch1)/PA3(ch2) · button=PB12 · LEDs 5/9/12/15V=PA4/PA5/PA6/PA7 (active-low, strap pads to GND) · spare=PB0/PB1.
- Reference sources: WCH EVT SDK is `WeActStudio.CH32X035CoreBoard/SDK/CH32X035EVT.ZIP` (extract to a temp dir; paths below are relative to `EVT/EXAM/`). PD example also at `WeActStudio.CH32X035CoreBoard/Examples/02-PD_Sink/`.

```c
/* Pick the highest source fixed-PDO voltage <= target_mv; PDO1 (5V) is the guaranteed floor. */
uint16_t pd_policy_select(uint16_t target_mv)
{
    uint16_t best = 5000;
    for (uint8_t i = 0; i < pd_source_pdo_count(); i++) {
        uint16_t mv = pd_source_pdo_mv(i);
        if (mv <= target_mv && mv > best) best = mv;
    }
    return best;
}
```

Request current = the selected PDO's advertised max (read from the PDO, not hard-coded).
